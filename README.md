# GPX-Math-Library

Simple and lightweight Math Library for graphics programming.


# Build And Linking 

run command to build static and dynamic linkable file "GPX_Math_Static", "GPX_Math_Dynamic"
```
 cmake /project/directory
 make
```
or
```
 cmake .. && make
```
